# Global Bank Statement Analyzer  

A **super easy tool** that reads your bank statement PDF (even if it's password protected) and instantly shows:

- Total Income  
- Total Expenses  
- Net Savings  
- Where you spent most (Food, Shopping, Travel, etc.)  
- Beautiful charts  
- Download full report  

Works with **SBI, HDFC, ICICI, Axis, Kotak, Yes Bank** and almost all Indian banks!

---

### Features
- Upload any bank PDF (password protected also works)  
- No need to type anything - just upload & click  
- Automatically detects columns (Date, Amount, Description, etc.)

- Smart spending categories (Food, Shopping, Bills, etc.)  
- Interactive charts (Pie, Bar, Line)  
- Download full data as CSV
